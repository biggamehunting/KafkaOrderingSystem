# KafkaOrderingSystem
# Order Processing System

## Tech Stack
- Java, Spring Boot
- Kafka (event-driven architecture)
- Microservices

## Architecture
Order Service → Kafka → Payment Service → Notification Service

## How to Run
1. Start Kafka
2. Run all services
3. Call POST /order

## Features
- Asynchronous processing using Kafka
- Scalable microservices design
